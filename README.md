# Spring Data JPA
This is a scalable backend spring data application that builds on postgreSQL.

Backend database is built on the following tables:
```
1️⃣ - Student
2️⃣ - StudentIdCard
3️⃣ - Book
4️⃣ - Courses
5️⃣ - Enrolment
```
